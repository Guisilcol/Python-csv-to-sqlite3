*** Esse projeto usa pipenv ***

Ao executar o programa, serão criadas duas pastas no mesmo diretório da execução do programa: 
    - inputFolder: pasta onde serão inseridos os arquivos de input para procesamento
    - outputFolder: pasta onde sesão armazenados os arquivos processados 

Lembre-se: a primeira linha de dados sempre deve conter os nomes das colunas.

Por padrão, o separador será ";".
Caso queira mudar o delimitador de colunas do arquivo, adicionar a seguinte estrutra no COMEÇO do arquivo: 

/*
sep=delimiter
*/
